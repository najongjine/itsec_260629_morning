
인터페이스: 연결해주다, 사용하게 해준다, 사용방법

p118
인터페이스 예를들면 어떤게 있느냐?
- DB Link: A DB에서 B DB에 있는 테이블을 마치 A DB 안에 있는 테이블처럼 접근하게 해주는 연결 기능
- DB Connection: 서버(WAS) 에서 DB에 접속하는 걸 지원해주는놈
- API: 응용 프로그램에서 다른 프로그램의 기능이나 데이터를 사용할 수 있도록 제공하는 인터페이스
   예: A 라는 서버에서 데이터를 받기 위해선 B 클라이언트는 {Get method 정보, token header 정보, 뭘 원하는지 목적}을 제공해야한다
- Socket : 프로그램이 네트워크로 데이터를 주고받기 위해 사용하는 통신 창구
   TCP/UDP 와 관련있음
   내 카카오톡 앱  →  네트워크  →  카카오톡 서버
   이때 내 앱과 서버가 데이터를 주고받으려면 각각 통신용 입구가 필요하다.
   그 입구가 소켓이다.
   
   소켓은 IP + Port와 관련 있다
   IP     = 건물 | 컴퓨터 주소
   Port   = 프로그램 번호
   Socket = 실제 통신하는 통로
